\page user_difficulty_ratings Difficulty Ratings

Rock Hero rates each arrangement on a 1-10 scale. The number is the source of truth; the
difficulty name is a display tier derived from that number.

| Rating | Difficulty |
|--------|------------|
| 0 | Unknown |
| 1-2 | Easy |
| 3-4 | Medium |
| 5-6 | Hard |
| 7-8 | Expert |
| 9-10 | Master |

Ratings belong to arrangements, not whole songs. A song can have separate lead, rhythm, and bass
arrangements, and each arrangement can have its own difficulty because each one may ask the player
to perform different techniques.

The Unknown value is reserved for unrated or unfinished arrangements. Songs intended for normal
play should use a rating from 1 through 10.
